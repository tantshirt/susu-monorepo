# DM template — Twitter / Discord

Use this when the partner is reachable through Twitter DM or a Discord channel rather than email. Keep it to one or two sentences plus the repo link.

## Variant A — generic

> Hey {{recipient_name}} — built Susu, an open-source Solana ROSCA primitive that uses {{partner_thing}}. Runnable example + 60s demo here: https://github.com/tantshirt/susu-monorepo — would love a tweet or doc-link if it resonates.

## Variant B — example-first

> Just shipped a runnable {{partner_name}} integration in Susu (open-source Solana ROSCA): https://github.com/tantshirt/susu-monorepo — happy to walk you through it. A tweet or reference link would unlock the "ecosystem partner" success criterion for our submission.

Replace `{{recipient_name}}`, `{{partner_thing}}` (e.g. "Privy embedded wallets", "Squads vault PDAs as group creator", "Helius RPC", "Token-2022 Transfer Hook + Metadata Pointer + Permanent Delegate"), and `{{partner_name}}` before sending.
