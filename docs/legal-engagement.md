## Day 1 Workflow

1. Send the firm the non-confidential SOW summary at `docs/legal-sow-summary.md`.
2. Ask the firm to sign a narrow T+0 SOW limited to the current non-custodial / non-fee / non-yield posture.
3. Run `bash scripts/legal-handoff.sh` to create the transient handoff bundle under `output_susu/legal-handoff/`.
4. Send counsel the bundle and confirm receipt in the daily log.
5. Replace `docs/legal-opinion.pdf` with counsel's signed letter if it arrives before submission close.
6. If the signed letter is delayed, keep the placeholder PDF generated by `bash scripts/render-legal-placeholder.sh`.

## Publication Rules

`docs/legal-opinion.pdf` is the only public legal-opinion publication path. Until a signed firm letter replaces it, the file must remain a placeholder that says the firm letter is delayed, links to `docs/legal-sow-summary.md`, and does not state any legal conclusion.

The engineering framing in `docs/fincen-cvc-framing.md` remains background only. Counsel's signed letter controls any legal conclusion once published.
