Susu Protocol — Color Theme

The committed direction is Phantom-fintech dark, with Solana mint-green as primary (not purple). Toggle the skin above to see how the same components render with the heritage (diaspora) palette — same dark base, copper/terracotta accents, but the mint --signal color stays for protocol-identity tokens (audit badge, on-chain checkmark, cluster pill).
